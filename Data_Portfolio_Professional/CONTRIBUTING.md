# Contributing

Want to add a new dataset or improve existing ones?

## How to Contribute

1. **Fork** this repository
2. **Create a new folder** for your dataset
3. **Include these files**:
   - dataset_name.csv (the data)
   - dataset_name.xlsx (Excel version)
   - SUMMARY.md (description)
   - preview.html (web dashboard)
4. **Submit a Pull Request**

## Dataset Requirements

- Must be realistic business data
- Include at least 100 records
- Have calculated business metrics
- Be clean (no missing values)
- Include proper documentation

## Questions?

Open an issue on GitHub or contact the maintainer.
